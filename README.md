# LibraryFrontend
Descrição: Foi utilizado para o desenvolvimento Angular 11.0.2, bootstrap e imagens aleatórias da API 'picsum.photos'.

## Start 

1. Instalar Depedências:
   
    npm install
   

2. Executar o Projeto:
   
    ng serve
   

## Build 
  
    npm run build

    ou

    ng build --prod
